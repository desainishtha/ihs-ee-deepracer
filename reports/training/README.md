This folder captures two supervised training evaluations performed in deepracer-for-cloud on macOS.

## Included Reports

- 30-minute run report: reports/training/30min_report.md
- 60-minute run report: reports/training/60min_report.md

- reports/training/metrics/TrainingMetrics_30min.json
- reports/training/metrics/TrainingMetrics_60min_attempt.json

Raw container logs:

- reports/training/logs/60min_algo_container.log

## Outcome Summary

Both runs were assessed as NO-GO for first physical-car deployment due to low completion percentages and high off-track rates.
